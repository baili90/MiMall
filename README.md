## 仿小米商场

#### 介绍
1.通过 Vue Router 进行路由封装，使用 Axios 发送与拦截网络请求。

2.通过 Vuex 实现组件之间的数据存储与共享，用于保存用户信息等。

3.根据接口文档，实现支付功能（支付宝、微信支付）的前端部分。

4.通过配置 Nignx 服务器，完成项目的线上部署，并用接口代理实现跨域。
